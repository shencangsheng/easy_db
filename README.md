# EasyDB

“开箱即用”，使用 SQL 查询 CSV、JSON、Parquet 

## 📖 Features

- CSV、JSON 文件以 SQL 的方式访问
- 查询结果导出 CSV、JSON 文件

## 🔮 Roadmap

- [ ] 优化异常提示

- [ ] 支持输出更多数据类型
